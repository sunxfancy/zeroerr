# function PrinterExt

template <typename T >
void PrinterExt(
    Printer & P,
    T v,
    unsigned level,
    const char * lb,
    rank< 0 > 
)

PrinterExt is an extension of Printer that allows user to write custom rules for printing.

Parameters:

  • P Printer class
  • v the object to be printed.
  • level the indentation level.
  • lb the line break.
  • r the rank of the rule. 0 is lowest priority.

Template Parameters:

  • T the type of the object to be printed.

User can use SFINAE to extend PrinterExt, e.g.:

template<typename T>
typename std::enable_if<std::is_base_of<llvm::Function, T>::value, void>::type
PrinterExt(Printer& P, T* s, unsigned level, const char* lb, rank<3>);
